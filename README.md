# Rankgun x Discord Bot Integration

Don't have RankGun Elite? Want to set users' ranks through your Discord bot? This is the repo for you!
In the file attached, you can integrate this directly into your Discord bot for easy ranking through Rankgun!
Created by: @kaiinspired (14.kaiinspired@gmail.com)

## Setup Instructions

1 - Place the "rankgun-bot.js" file into your bot.
2 - Open the file and configure your variables, either in the script or in your .env
3 - Deploy and test it!
